เชื่อมต่อ Google Sheet
ทำตาม 4 ขั้นตอนด้านล่าง เพื่อเชื่อมระบบกับ Google Sheet ของคุณ
1
เปิด Google Sheet ใหม่ → เมนู Extensions → Apps Script
2
วางโค้ดจากไฟล์ Code.gs ที่ได้รับ แทนที่โค้ดเดิมทั้งหมด → กด Ctrl+S
3
คลิก Deploy → New Deployment → Type: Web App → Who has access: Anyone → กด Deploy → อนุญาต Permission
4
Copy Web App URL ที่ได้ มาวางด้านล่าง แล้วกด เชื่อมต่อ
ข้อมูลทั้งหมดจะถูกเก็บใน Google Sheet ของคุณโดยตรง
